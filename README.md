# 📊 Statistical Business Analysis Project

## 📌 Project Overview
This project performs statistical analysis on business sales data to generate meaningful insights.

Objectives:
- Perform descriptive statistics
- Analyze data distribution
- Conduct hypothesis testing
- Calculate confidence intervals
- Perform regression analysis
- Generate actionable business insights

---

## 🛠️ Setup Instructions

1. Clone the repository
2. Install dependencies:

pip install -r requirements.txt

3. Run Jupyter Notebook:

jupyter notebook statistical_analysis.ipynb

---

## 📂 Project Structure

- statistical_analysis.ipynb → Main analysis
- sales_data.csv → Business dataset
- customer_churn.csv → Customer dataset
- hypothesis_tests_results.txt → Test outputs
- statistical_report.pdf → Final report
- requirements.txt → Required libraries

---

## 📊 Key Results (Sample)

- Average Sales: $45,200 ± $3,400 (95% CI)
- Correlation (Marketing-Sales): 0.78 (Strong Positive)
- p-value: 0.0012 → Significant
- R-squared: 0.61

---

## 📈 Business Insights

- Marketing spend significantly increases sales
- Strong positive relationship between marketing and revenue
- Business should increase marketing budget strategically

---

## 🧪 Hypothesis Tests Conducted

1. Pearson Correlation Test
2. Independent T-Test
3. ANOVA Test
